# PCCM OEM Launcher

An OEM-inspired launcher for Porsche Android head units.

The goal of this project is to provide a clean, factory-style interface that closely resembles Porsche's original PCCM+ user experience while retaining the flexibility of Android.

Unlike many aftermarket launchers, PCCM OEM Launcher focuses on simplicity, consistency, and an OEM appearance rather than excessive customization.

## Features

- OEM-inspired Porsche PCCM+ interface
- Clean grayscale icon set
- User-configurable application shortcuts
- Long-press any tile to assign a different application
- Lightweight and fast
- Designed specifically for automotive use
- No unnecessary widgets or distractions

---

## Compatibility

This launcher has been tested on:

- UIS7870-based Android head unit
- Samsung Galaxy Tab S9 FE
- BlueStacks 

Other android devices should also work, but compatibility is not guaranteed.

Android head units vary significantly between manufacturers, firmware versions, and ROMs. The method used to configure a default launcher differs between devices. If you are unsure on how to change your launcher, please consult your head unit manufacturer for model-specific instructions.

---

## Installation

1. Download the latest APK from the Releases page.
2. Copy the APK to a USB drive.
3. Install the APK on your Android head unit.
4. Configure it as your preferred launcher according to your device manufacturer's instructions.
5. Long-press any icon to assign the application you want that tile to launch.
